《深入浅出统计学》总结

发表于 2019-06-05 | 分类于 统计分析 | | 本文总阅读量 次

1、集中趋势的度量

  • 均值 $\mu = \frac{\sum x}{n}$。

  • 频数 $\mu = \frac {\sum fx}{f}$, f是频数。

  • 中位数：1）排列组合为奇数时，$(n+1)/2 $；2）为偶数时，为两个中间数相加除以2。

  • 众数

2、分散性与变异性的量度

  • 全距：上界-下届 (最大值 — 最小值)
    • 展示数据分布的一种简单情况，但有局限性
    • 容易受异常值（大或小）的影响

  • 四分位数：将数据从小到大排列后一分为四

    • 下四分位数Q1

      • n/4=整数，则Q1为n/4这个位置与上一个数字的平均值
        • n/4=不为整数，则Q1取n/4的向上取整

    • 上四分位数Q2

      • 3n/4=整数，则Q1为3n/4这个位置与上一个数字的平均值
      • 3n/4=不为整数，则Q1取3n/4的向上取整

    • 四分位距=上四分位数-下四分位数

      • 优点：较少受到异常值的影响，剔除异常值

  • 百分位数：第K百分位数即位于数据范围k%的位置的数值Pk

    • 优点：对于划分档次非常有用

    • 计算百分位距与四分位数一样，k(n/100)

      • 若为整数，则取该位置数与上一个数取平均值
      • 若为小数，则向上整数值，则该数值即为百分位数
  • 箱线图：能在同一张图上体现多个距和四分位数
    • 箱：上下四分位数和四分位距
    • 线：上下界
    • 优点：能够体现多个数据，非常有利于比较
  • 方差：量度数据分散性的一种方法，是数值与均值距离的平方数的平均值
    • $\sigma^2 = \frac{\sum(x-\mu)^2}{n}$
    • 方差速算（另一种较为简便的算法）：$\sigma ^2 = \frac{\sum x^2}{n}-\mu^2$
  • 标准差：方差的开方
    • 优点：度量数值与平均数之间的距离，最小值为0
    • $\sigma = \sqrt{\sigma ^2}$
  • 标准分: 标准分是利用均值和标准差，将一个数据集中的各个数据转化为更通用的分布形态，同时确保数据的基本形态形状不变
    • $z = \frac{x-\mu}{\sigma}$
    • 是对不同数据集的数值进行比较的方法
    • 标准化后均值为0，标准差为1

3、概率

  • 事件概率：$p(A)=n(A)/n$，分为独立事件和相关事件

    • 相关事件下的互立事件：发生A则不发生B，$p(A)+p(B)=1$
    • 独立事件：相互平行的，$P(A|B)=P(A),P(AB)=P(A)*P(B)$

  • 条件概率：$P(B|A)=P(AB)/P(A)$

  • 全概率公式：根据条件概率，事件A或者B的概率

    • $P(B)=P(B|A’)P(A’)+P(B|A)P(A)$

  • 贝叶斯定律：根据条件概率和全概率，逆向求条件概率

    • $P(A|B)=P(B|A)P(A)/[P(B|A’)P(A’)+P(B|A)*P(A)]$

  • 以上全概率和贝叶斯概率，尤其是后者，画概率树最好使。

4、期望

概率分布：描述一个给定变量给出的所有可能的结果

  • 一个随机变量X期望等于我们所期望的长期平均值，用$E(X)$或者u表示

  • 期望：$E(X) = \sum xP(X=x)$

    • X的函数的期望，$E(f(X))=\sum f(x)P(X=x)$

  • 方差：$Var(X)=E(X-μ)²$

  • 标准差： $\sigma=\sqrt{Var(X)}$

  • 线性变换： eg：一台老虎机增大了赌本

    • 在变量中增加一个常数不过是将概率分布移动一下，分布形状不变
    • $E(aX+b)=aE(X)+b$
    • $Var(aX+b)=a²Var(X)$

  • 独立观测值：一个事件的结果是一个独立观测值，几个互不影响的事件的结果称为独立观测值 eg：老虎机每一次拉杆为一个独立观测值

    • X的独立观测值与X不同，每个独立观测值都具有相同的概率分布，但结果不一样
    • $X₁、X₂、…、Xn$是X的独立观测值
    • $E(X₁+X₂+…+Xn)=nE(X)$
    • $Var(X₁+X₂+…+Xn)=nVar(X)$

  • 独立随机变量：如果X和Y是独立随机变量 eg：两台老虎机

    • $E(X±Y)=E(X)±E(Y)$
    • $Var(X±Y)=Var(X)+Var(Y)$
    • 线性变化
    • $E(aX±bY)=aE(X)±bE(Y)$
    • $Var(aX±bY)=a²Var(X)+b²Var(Y)$

5、排列组合

  • 排位方式的计算公式

    • 如果要求N各对象的 可能排位方式的数目，则计算：
      • $n! = n(n-1)…32*1$
      • 也就是说，将从n到1的数字全部相乘
      • 如果n个对象昨圆形排位，则可能的排位方式数目是$(n-1)!$

  • 排列是指从一个较大（n个）对象群体中取出一定数目（r个）对象进行排序，并得出排序方式总数目

    • $P_r^n = \frac{n!}{(n-r)!}$
    • 排列指从群体中选取几个对象，在考虑这几个对象的顺序的情况下，求出这几个对象的选取方式的数目
    • 也即：排列与顺序有关

  • 组合指从一个群体中选取几个对象，在不考虑这几个对象的顺序的情况下，求出这几个对象的选取方式的数目

    • $C_r^n = \frac{n!}{r!(n-r)!}$
    • 组合是与顺序无关的

6、几何分布、二项式分布及泊松分布

几何分布

  • 何时使用

    • 进行一系列独立试验
    • 每次实验都存在成功或者失败的可能
    • 主要想知道的是为了取得第一次成功需要实验多少次

  • 如果符合几何分布条件，则：X~Geo(p) ,p代表单次实验的成功率，q=1-p，X是为了获得第一次成功进行实验的次数

  • 概率

    • 在第r次试验时取得第一次成功的概率：$P(X=r) = q^{r-1}p$
    • 需要进行r次以上实验（失败）才能取得第一次成功的概率：$P(X>r)= q^r$
    • 需要进行r次或不到r次实验即能取得第一次成功的概率：$P(X≤r)=1-q^r$

  • 方差和期望

    • $E(X)= \frac{1}{p}$
    • $Var(X)= \frac{q}{p²}$

二项式分布

  • 何时使用

    • 进行一系列次数有限的独立试验
    • 每次实验都存在成功或者失败的可能
    • 主要想知道的是成功或失败的次数

  • 如果符合二项式分布，则：X~B(n,p) p代表单次实验的成功率，q=1-p，n为实验次数，X为在n次实验中取得的成功次数

  • 概率

    • $P(X=r)=ⁿC_{r}× p^r × q^{n-r}$
    • 其中$ⁿC_{r}= \frac{n!}{r!(n-r)!}$

  • 方差和期望

    • $E(X)=np$
    • $Var(X)=npq$

泊松分布

  • 何时使用

  • 单个事件在给定区间内随机、独立地发生

    • 已知给定区间内的事件平均发生次数，或者叫发生率，且这个次数或发生率是有限的
    • 主要想知道的是给定区间内的发生次数

  • 如果符合泊松分布，则：X~Po(λ) λ代表发生率，X表示给定区间内的事件发生次数，

  • 概率

    • $P(X=r)= \frac{e^{λ}λ^r}{r!}$

  • 方差和期望

    • $E(X)=λ$
    • $Var(X)=λ$

  • 如果$X \sim Po(λx)$，$Y\sim Po(λy)$，且X和Y是独立的，则：$X+Y\sim Po(λx+λy)$

  • 如果$X\sim B(n,p)$，其中n足够大（比如大于50），p足够小（比如小于0.1），则可以将二项分部近似看作泊松分布$X\sim Po(np)$

7、正态分布的应用：保持正态

  • 离散型概率分布：前面讲到的都是离散数据，即数据是由一个个的数值组成，其中每个数值都有相应的概率。

  • 连续性概率分布：连续数据往往通过测量得到，而不是通过计数得到，测量结果在很大程度上取决于测量精度的要求。

    • 连续概率分布：用概率密度函数进行描述
    • 计算一个数值范围内的概率：$P(a<x<b）=a到b与f(x)组成的面积$
    • 概率密度函数下方的总面积必须等于1.
    • 正态分布
      • $X\sim N(\mu,\sigma2)$
      • 计算某一数值的正态分布概率，需要进行标准化，$\frac{X-\mu}{\sigma}$ ,进而查出概率分布，通过表格 $X\sim N(0,1)$ 的正太分布规律。

再谈正态分布的运用：超越正态

体重，身高之类的数据属于连续性数据，往往符合正态分布，而他们的综合数据（加或者减也都符合正态分布，这时候就需要用到之前学到的离散型数据的公式计算连续性数据综合数据）

  • 超越正态1—n个独立变量的综合数据计算

  • $X\sim N(\mu_{X},\sigma_{X}^{2})$ ,$Y\sim N(\mu_{Y},\sigma_{Y}^{2})$为两个独立变量

    • 线性变化：影响概率分布中的基本数值，例如，如果你有一根特定长度的绳子，那么，进行线性变化会影响绳子的长度。如果基本数据发生变化，则所面对的就是变换
      • $X\sim N(\mu,\sigma^{2})\rightarrow aX+b\sim N(a\mu+b,a^{2}\sigma^{2})$

    • 独立观察结果：影响所处理事件的数量，例如，如果一段绳子有n个独立观察的结果，则所讨论的就是n段绳子。如果数量发生变化，则面对的就是独立变量

      • $X+Y\sim N(\mu_{X}+\mu_{Y},\sigma_{X}^{2}+\sigma_{Y}^{2})$

      • $X-Y\sim N(\mu_{X}-\mu_{Y},\sigma_{X}^{2}+\sigma_{Y}^{2})$

      • 如果$X1,X2,X3…Xn$为独立观察结果，且 $X\sim N(\mu,\sigma2)$

        • $X_{1}+X_{2}+X_{3}+…+X_{n}\sim N(n\mu,n\sigma2)$

超越正态2—正态分布近似代替二项式分布（B-N(n,p)）

条件：如果np和nq双双大于5时，可以用正态分布近似代替二项分布，即 $X\sim N(np,npq)$ *;如果n>50且p<0.1,则可以使用泊松分布代替二项分布。

记住：利用正态分布代替二项分布时候，必须进行连续性修正。

  • 求解过程：

    • 求解均值和方差：B-N(n,p)，均值np，方差npq；

    • 转化成 $X\sim N(np,npq)$

    • 利用标准分计算概率

    • 问题：需要进行修整，因为正态分布是连续性数据，而二项式分布是离散型数据，才能确保正确的结果。

      • $\leq$ 型概率求解：$P（X \leq a） \rightarrow$ 实际计算正态分布下的P（X <a+0.5）
      • $\geq$型概率求解：$P（X \geq b）\rightarrow$ 实际计算正态分布下的P（X > b-0.5）
      • 介于型概率求解：$P（ a\leq X \leq b）\rightarrow$ 实际计算正态分布下的P（ a-0.5< X < b+0.5）
      • X=a概率性求解：$P（X=a）\rightarrow$ 实际计算正态分布下的P（ a-0.5< X < a+0.5）

超越正态3—正态分布近似代替泊松分布$（X-P0(\lambda)）$

条件：如果$X \sim P0(\lambda)$且$\lambda >15$,则可以使用正态分布代替泊松分布，$X\sim N(\lambda,\lambda)$

但是：别忘记进行连续性修正，修正方式与上述二项式分布类似。

9、总体和样本的估计：进行预测

  • 估计总体均值：样本均值（ $\bar{x}$）被称为总体均值（ $\mu$ ）的点估计量（$\bar{\mu}$ ）

  • 估计总体方差：$\bar{\sigma^{2}}=s^{2}=\frac{\sum_{a}^{b}{（X-\bar{X}）^{2}}}{n-1}$

  • 预测总体比例： $\bar{p}=\frac{成功样本}{总体样本}$

  • 比例的抽样分布：从一个总体中取得所有大小为n的可能样本，由这些样本的比例形成一个分布，形成比例的抽样分布。我们用 $P_{s}$ 代表样本比例随机变量

    • $P_{s}$ 的期望和方差定义：

      • $E（P_{s}）=p$
      • $Var（P_{s}）=pq/n$
      • $比例标准误差= \sqrt{Var(ps)}$
      • 其中，p指的是总体中某一性质的变量所占的总体比例,n指的是样本总量

    • 如果n>30，则 $P_{s}$ 符合正态分布，于是 $P_{s}\sim N(p,pq/n)$ ,但是需要进行连续性修正，即上述进行修正的时候的0.5变成+-1/2n

      • $P(P_{s}\geq0.4) = P(P_{s}\geq(0.4-\frac{1}{2*n}))$
      • $P(P_{s}\leq0.4) = P(P_{s}\leq(0.4+\frac{1}{2*n}))$

  • 均值的抽样分布：样本均值分布

    • 首先：总体X分布，X的均值为 $\mu$ ，方差为 $\sigma^{2}$，因此$E（X）= \mu$，$Var(X)= \sigma^{2}$

    • 其次：用来自 总体X的所有大小为n的可能样本，形成所有样本均值的分布—— $\bar{X}$的分布，这个分布的均值和方差如下：

      • $E（X）= \mu$
      • $Var(X)= \sigma^{2}/n$
      • $均值标准误差= \sigma/\sqrt{n}$
      • n越大，均值标准误差越小，样本个体数越多，作为总体均值的估计量的样本均值越可靠。

    • 计算样本均值 $\bar{X}$的分布：

      • 若总体X符合正态分布，则无论样本n有多大，均值的分布都符合正态分布：

        • $\bar{X}\sim N（\mu，\frac{\sigma^{2}}{n}）$

      • 若总体X不符合正态分布，则样本n较大时候，可以认为X符合正态分布 $\bar{X}$ 的分布近似正态分布，即中心极限定理。

  • 中心极限定理：如果从一个非正正态总体X取出一个样本，且样本很大，则 $\bar{X}$的分布近似正态分布，即：总体X分布，X的均值为 $\mu$，方差为 $\sigma^{2}$，且样本n很大时候：

    • $\bar{X}\sim N（\mu，\frac{\sigma^{2}}{n}）$

      • 二项分布： $X\sim N(n,p)$ , 均值： $mu=np$;方差：$\sigma^{2}=npq$; 当n>30,根据中心极限定理 $\bar{X}\sim N（np，pq）$

      • 泊松分布：$ X\sim N(n,p) $,$\mu=\sigma^{2}=\lambda$;当当n>30,根据中心极限定理$\bar{X}\sim N（\lambda，\frac{\lambda}{n}）$

      • 最后，按照正态分布标准化算法计算概率

      注意：中心极限定理不需要进行修正

10、置信区间的构建：自信地猜测

当n较大，可以近似看成是正态分布

  • $\mu$的置信区间： $p(a<\mu<b)=0.95$,总体均值介于a和b之间，则（a,b）被称为置信区间

    • 1、选择总体统计量：总体均值E(X)= $\mu$，总体方差为 $Var(X)=\sigma^{2}$

      • 2、求出所选统计量的抽样分布：均值的抽样分布，其均值为$E( \bar{X} )= \mu$ ,均值方差为$Var( \bar{X})=\sigma^{2}/n$

        • 问题是：总体方差 $\sigma^{2}$未知，因此需要样本方差$S^{2}$来代替
          • 因此可以求出均值抽样分布： $\bar{X}\sim N（\mu，\frac{S^{2}}{n}）$

      • 3、决定置信水平：一般是0.90或者0.95

      • 4、求出置信上下限：标准化计算 ： $Z=\frac{\bar{X}-\mu}{s/\sqrt{n}}\sim N(0,1)$ ,查表得到置信区间对应值，则就可以改写成 $\mu$关于 $\bar{X}$的范围值，其中 $\bar{X}$是样本计算的均值 $\bar{x}$

当n较小，可以近似看成是t分布

  • 1、选择总体统计量

  • 2、均值抽样分布 $\bar{X}$的概率分布：$T \sim t(v)$ ,其中 v是自由度=n-1,n为选取的样本数

    • t分布的标准分： $T=\frac{\bar{X}-\mu}{s/\sqrt{n}}$

  • 3、决定置信水平：置信水平能得出标准分下的T的上下限值所对应的范围，根据置信概率和自由度v来确定值。

  • 4、求出置信上下限：即 $\mu$的范围。

无论求置信区间是正态分布还是t分布，不可缺少的是确定该样本符合什么类型的分布，二项式分布，泊松分布，还是等等，这样有利于得出方差$\sigma^{2}$ 是何值

有两种估计总体统计量的方法

  • 使用点估计量：可估计精确数值
  • 使用总体统计量的置信区间，得到一个有较高可信度的数值范围

11、假设检验的运用：研究证据

假设检验6步骤：

  • 1）确定要进行检验的假设

    • H0：原假设即你要对其进行检验的断言，除非有足够的证据进行反驳，否则你将接受这个断言
    • H1：备择假设即在拒绝H0之后将接受的另一个断言
    • 进行假设检验时，你假定原假设为真；如果有足够的证据反驳原假设，则拒绝原假设，接受备择假设
  • 2）选择检验统计量
    • 检验统计量：即用于对假设进行检验的统计量，是与该检验关系最为密切的统计量
  • 3）确定拒绝域
    • 显著性水平：希望在样本结果的不可能程度达到多达时，就拒绝原假设H0。
    • 单尾检验：检验的拒绝域落在可能的数据集的一侧
    • 双尾检验：拒绝域一分为二在数据集的两侧
  • 4）求出p值
  • 5）样本结果位于拒绝域中吗
  • 6）作出判断

案例：

某药物厂家声明某新药物的治愈率可达90%。医生对新药物的实验结果：

是否治愈？ 是 否
频数 80 20

  • 第一步：确定假设

    • H0：药物治愈率达到90%，即 p = 0.9
    • H1：药物治愈率低于90%，即 p< 0.9

  • 第二步：选择检验统计量

    • 样本中的患者数量是100，治愈的患者的概率为0.9，服从二项分布。即：X ~ B(100, 0.9)
    • 而计算量大，我们可以用正态分布进行代替，其中n=100，p=0.9。（由于n很大，且np和nq都大于5，因此检验统计量服从 X ~ N(90, 9)
    • 标准化 $z = \frac{x-90}{\sqrt{9}}$， z ~ N(0, 1)

  • 第三步：确定拒绝域

    • 由于备择假设p < 0.9，表明拒绝域在左尾
    • 以5%为拒绝域，查标准正态分布，临界值c等于$P(Z < c) = 0.05$，即$ c = -1.64$
    • 说明只要检验统计量小于-1.64，我们就有足够的证据拒绝原假设
  • 第四步：求p值
    • 拒绝域位于分布的左尾，治愈人数为80人，$Z = (X - 90)/3$，利用这些条件求P值
    • 先求80的标准分，$ z = (80 - 90)/3 = -3.33$
    • p值算法为$P(Z < z) = P(Z < -3.33)$，查概率表，得：$p = 0.0004$
  • 第五步：查看检验统计量是否位于拒绝域内
    • 如果p值小于0.05，则检验统计量位于拒绝域内。由于p值等于0.0004，说明检验统计量位于拒绝域内。
  • 第六步：做出决策
    • 由于检验统计量位于假设检验的拒绝域中，说明5%显著性水平的情况下，我们有足够的证据拒绝原假设，即接受“药物治愈率低于90%”。

12、$\chi^2$分布

检验统计量$\chi^2 $提供了一种对观察频数和期望频数之间的差异进行度量的办法。$\chi^2$值越小，观察频数和期望频数之间的总差值越小。

  • $\chi^2 = \sum \frac{(O-E)^2}{E}$ O代表观察频数，E代表期望频数

  • 两个用途：

    • 第一是用于检验拟合优度，也就是可以检验一组给定的数据与指定分布的吻合程度；
    • 第二是检验两个变量的独立性，通过这个方法可以检查变量之间是否存在某种关联。

  • 卡方分布： $X^2 = \chi^2(\nu)$

    • 其中，$\nu$是卡方参数，自由度；
    • 卡方分布进行的检验是单尾检验，右尾被作为拒绝域；
    • 如果用显著性水平$\alpha$进行检验，则 $\chi^2 _\alpha(\nu)$ 。
